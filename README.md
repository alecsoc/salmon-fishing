# Pescar Al Salmón 🐟
Minijuego desarrollado en Python con la librería Pygame. Este proyecto forma parte de la cátedra de Objetos y Abstracción de Datos (Grupo 10).

## 📝 Descripción
Explora las profundidades y pon a prueba tus reflejos. Utiliza el sonar para detectar bancos de peces en la oscuridad y tu linterna para capturar los objetivos antes de que se agote el tiempo.

El juego desafía la gestión de recursos visuales y la velocidad de reacción a través de tres niveles de dificultad.
